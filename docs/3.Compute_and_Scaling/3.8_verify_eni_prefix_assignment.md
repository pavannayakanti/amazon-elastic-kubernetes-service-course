# Verifying ENI and Prefix Assignment in AWS EKS

## Step 1: Create a new Pod

Deploy the pod using the same YAML as in the previous lab (nginx).

```bash
cat <<EOF | kubectl apply -f -
apiVersion: v1
kind: Pod
metadata:
  name: nginx
  labels:
    app: nginx
spec:
  containers:
  - name: nginx
    image: nginx:latest
    ports:
    - containerPort: 80
EOF
```
